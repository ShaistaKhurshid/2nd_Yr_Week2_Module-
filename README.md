# 📚 2nd Year - Week 2 Module

## 📅 Date
17/06/2026

## 📖 Overview

This repository contains solutions for Week 2 Data Structures & Algorithms lab exercises. The module focuses on fundamental Searching and Sorting Algorithms used for efficient data processing and problem-solving.

The programs included demonstrate:

- Linear Search
- Binary Search
- Bubble Sort
- Insertion Sort
- Selection Sort

---

## 📝 Programs Included

### 1. Find Student Roll Number (Linear Search)

Searches for a student roll number in an unsorted list using the Linear Search algorithm.

**Concepts Covered:**
- Sequential Search
- Array Traversal
- Search Operations

---

### 2. Search Product ID (Binary Search)

Searches for a product ID in a sorted list using the Binary Search algorithm.

**Concepts Covered:**
- Divide and Conquer
- Efficient Searching
- Sorted Arrays

---

### 3. Arrange Exam Scores (Bubble Sort)

Sorts students' exam scores in ascending order using Bubble Sort.

**Concepts Covered:**
- Adjacent Element Swapping
- Sorting Techniques
- Nested Loops

---

### 4. Arrange Library Book Numbers (Insertion Sort)

Arranges book serial numbers in ascending order using Insertion Sort.

**Concepts Covered:**
- Incremental Sorting
- Array Manipulation
- Ordered Insertion

---

### 5. Rank Players by Score (Selection Sort)

Ranks player scores from lowest to highest using Selection Sort.

**Concepts Covered:**
- Minimum Element Selection
- Sorting Algorithms
- Array Processing

---

## 🎯 Learning Outcomes

After completing this module, students will be able to:

- Understand searching techniques and their applications.
- Differentiate between Linear Search and Binary Search.
- Implement common sorting algorithms.
- Analyze algorithm efficiency.
- Apply sorting and searching techniques to real-world problems.
